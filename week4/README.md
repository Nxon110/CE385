1. ฟังก์ชัน operation รับชนิดการคำนวณ (type) และตัวเลข a b จากนั้นตรวจสอบ type เพื่อคำนวณ บวกลบคูณหาร และส่งผลลัพธ์กลับมา ถ้าไม่ตรงจะคืนค่า "Invalid operation type"

2. ฟังก์ชันการทำงานแบบ asynchronous แบบ Callback พอเวลาครบแล้ว จะเรียกฟังก์ชันกลับมาเพื่อบอกผลลัพธ์ว่าได้ข้อมูลหรือ error แบบ Promise รอผลลัพธ์แล้วค่อยไปทำต่อใน .then() ถ้าพังจะเข้า .catch()

3. ฟังก์ชันนี้การทำงานโดยใช้ Promise และ async/await ถ้าเวลาที่ส่งเข้าไป มากกว่า 1000 ms งานจะสำเร็จและแสดงผลลัพธ์ ถ้าเวลาน้อยกว่า 1000 ms จะถือว่าพลาด แล้วเข้า catch แสดง error

4. เป็นการเรียกข้อมูลจากเซิร์ฟเวอร์ 3 แหล่งแบบ Asynchronous โดยใช้ Promise Promise.any จะเลือกผลลัพธ์ที่ สำเร็จเป็นตัวแรก เเละ Promise.allSettled จะแสดงผลลัพธ์ของ ทุกเซิร์ฟเวอร์ ไม่ว่าจะสำเร็จหรือเกิดข้อผิดพลาดก็ตาม
