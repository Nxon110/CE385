1.1 อันดับเเรกก็คือการประกาศตัวเเปรใช้ let ในการประกาศ ชื่อ,อายุ,สาขา เเล้วใช้ colsole เพื่อใช้เเสดงผลลัพธ์ออกมา

1.2 ประกาศตัวเเปรทั้ง 5 message,score,isPassed,emtyValue,unDefined เเล้วใช้ colsole เพื่อเเสดงผลลัพธ์ออกมา เเละใช้ typeof เพื่อเช็คว่าเป็น type จริงหรือไม่

1.3 ใช้ let ประกาศตัวเเปรก่อนอันดับเเรกวิชาเเละคะเเนน เเล้วจากนั้นใช้ let เพื่อรวมคะเเนนของทั้ง 3 วิชา เเละคำนวนค่าเฉลี่ยเป็นทศนิยม เเละคะเเนนมากกว่า 80 หรือไม่ เเละใช้ colsole เพื่อเเสดงค่าผลรวม
